# Rooyin Arabic Font (Minecraft 26.1.2)

This is the **Rooyin Arabic font** for Minecraft, re-packaged so it works on **Minecraft 26.1.2**.

The only change made is updating `pack.mcmeta` to the new pack format (`min_format`/`max_format: 84`) so the pack is recognized on 26.1.2. **No font or asset files were modified.**

## Installation

1. Download the `.zip` from the [Releases](../../releases) page.
2. Put it in your Minecraft `resourcepacks` folder.
3. In game: Options > Resource Packs, and enable it.

## Why this exists

This font pairs with the **Arabic Input Fix** mod, which fixes Arabic text rendering in Minecraft's chat input box. The mod handles correct letter connection and right-to-left order; this font makes the Arabic letters look clean instead of the default Minecraft Arabic font.
